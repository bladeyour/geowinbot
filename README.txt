
1. Залий усе на GitHub (новий репозиторій)
2. Перейди на https://render.com → New + → Web Service
3. Вибери GitHub репозиторій
4. У середовищі (Environment):
    - Add Environment Variable:
        - KEY: BOT_TOKEN
        - VALUE: твій токен Telegram бота
5. Установи:
    - Build Command: `pip install -r requirements.txt`
    - Start Command: `python3 main.py`
6. Готово! Бот працює 24/7
